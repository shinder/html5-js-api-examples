HTML5 API 學習 PWA

本頁示範最基本的 PWA — 只要有 manifest.webmanifest + 圖示，網站就「可安裝」。

用 Chrome / Edge 開啟此頁，網址列右側會出現「安裝」按鈕。

iOS Safari：點擊「分享」→「加到主畫面」。

注意：本頁僅含 manifest，沒有 Service Worker，所以「離線可用」尚未具備， Chrome 的網址列安裝圖示也未必會顯示。下一個範例會加上 Service Worker。
